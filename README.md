# Alpha Camp Rails習題︰Restaurant Forum

## 如何啟動？

```ruby
git clone git@github.com:victorloo/AC_to_do_list.git
bundle install
rake db:migrate
rails dev:fake_list
```

## 開發環境

* Ruby version: 2.4.4
* Rails version: 5.1.4

### 使用的 gem

* [ffaker](https://rubygems.org/gems/ffaker)
* [bootstrap-sass](https://rubygems.org/gems/bootstrap-sass)

## 使用者故事

* 用戶可以瀏覽所有代辦事項
* 用戶可以新增代辦事項
* 用戶可以瀏覽一筆代辦事項
* 用戶可以修改一筆代辦事項
* 用戶可以刪除一筆代辦事項
